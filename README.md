# PirateBox
Updated site for pirate boxes

# Installation

Install a standard Raspberry Pi pirate box. 

Setup PHP:

https://forum.piratebox.cc/read.php?7,16060

scp web files over:

sudo scp -r "localwwwfolder" "username"@"pirateboxaddress":/opt/piratebox/
